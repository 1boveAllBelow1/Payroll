# Payroll System

## How to use

1. Install nyo XAMPP.

2. Start nyo  Apache and MySQL servers.

3. lagay nyo ung  project folder sa loob ng  `htdocs` directory

4. Open phpMyAdmin sa `http://localhost/phpmyadmin/`tas gawa kayo database named `payroll_db`.

5. Import nyo `payroll_db.sql` sa  database.

6. Edit `connect.php` pag iba pangalan ng  database username or password name nyo.

7. Open browser punta sa: http://localhost/payroll_system/

8. hanapin nyo nlng sa database ung admin, employee ids and password lol.
